## StreamlitWebApp_of_SentimentAnalysis_of_IMDbMoviedata_using_RNN_Tensorflow

![image](https://www.panoramaaudiovisual.com/wp-content/uploads/2019/06/IMDB.jpg)

## Data
- The data is directly downloaded from tensorflow using the datasets in-built in the same
- The data is already pre-encoded

## Processing
- Minimal pre-processing is done with respect to the text as it is relatively clean

## Notebooks
- Consists of a [notebook](https://github.com/nandhika03/StreamlitWebApp_of_SentimentAnalysis_of_IMDbMoviedata_using_RNN_Tensorflow/blob/main/notebooks/RNN_IMDB.ipynb) that includes basic embedding process examples and also building of the RNN model using a sequential model with an embedding layer and one dense layer (output)
- This [notebook](https://github.com/nandhika03/StreamlitWebApp_of_SentimentAnalysis_of_IMDbMoviedata_using_RNN_Tensorflow/blob/main/notebooks/prediction_rnn.ipynb) consists of the prediction being done using the model saved as a pickle file.

## Models
- This [file](https://github.com/nandhika03/StreamlitWebApp_of_SentimentAnalysis_of_IMDbMoviedata_using_RNN_Tensorflow/blob/main/models/RNN_model_imdb.h5) consits of the saved RNN model

## App deployment
- The application is deployed in streamlit and the application file can be found [here](https://github.com/nandhika03/StreamlitWebApp_of_SentimentAnalysis_of_IMDbMoviedata_using_RNN_Tensorflow/blob/main/app.py)

- For all the required packages for this web app, refer to [this](https://github.com/nandhika03/StreamlitWebApp_of_SentimentAnalysis_of_IMDbMoviedata_using_RNN_Tensorflow/blob/main/requirements.txt) document
